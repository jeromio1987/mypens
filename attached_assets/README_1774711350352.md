# PENS — Holiday Folder

Everything you need to keep working on PENS from any device, with any LLM.

## Files in this folder

| File | Use it for |
|------|-----------|
| **README.md** | This file — orientation |
| **PENS_Brief.md** | Paste into any LLM at the start of a session to get it up to speed fast |
| **PENS_Vision.md** | Full product vision, philosophy, roadmap — the master doc |
| **PENS_WeightEngine.md** | Full logic spec for the weight interpretation engine |
| **PENS_Clubroom.md** | Narrative layer spec — card formats, tone, language bank |
| **PENS_Decisions.md** | Log of confirmed decisions + open questions + rejected ideas |
| **PENS_ReplitBrief.md** | Ready-to-use brief for Replit — feature specs with function signatures |

---

## How to use these with each LLM

### Claude (strategy, spec, decisions)
Paste PENS_Brief.md at the start. For deep work, paste PENS_Vision.md. Claude will remember context within the session and update files when decisions are made.

### GPT (copy, brainstorming, card text)
Paste PENS_Brief.md. Ask for copy variations, card headlines, tagline options, onboarding text. Bring winners back to Claude to log in PENS_Decisions.md.

### Replit (building)
Paste PENS_ReplitBrief.md and the specific feature section you're building. Never give Replit a vague brief — spec it here first, then hand off.

---

## One rule
When something is decided, it goes in PENS_Decisions.md before the session ends. That's how context survives across devices, LLMs, and holidays.
